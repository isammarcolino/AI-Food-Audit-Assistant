## 🚀 Como Executar (Google Colab)

1. Abra o [notebook Analise_Completa.ipynb](notebooks/Analise_Completa.ipynb)
2. Execute todas as células
3. O modelo Qwen2.5 será baixado automaticamente
4. Relatórios são gerados em `outputs/`

## 🛠️ Tecnologias

- Python 3.10+
- Pandas (análise)
- Matplotlib (gráficos)
- llama.cpp + GGUF (IA local)
- Qwen2.5-3B-Instruct (modelo)
- Google Colab (execução)

Arquivo	Motivo

config.py	Centraliza configurações. Evita erros por caminhos errados.
utils.py	Funções reutilizáveis (download, logging). Separa responsabilidades.
prompts.py	Mantém prompts limpos e editáveis. Crucial para qualidade da IA.
data/	Local padrão para entrada. Facilita versionamento.
outputs/	Saída organizada. Evita poluição do repositório.
notebooks/	Didática. Permite passo a passo para estudantes.
README.md	Portfólio profissional. Explica o projeto em minutos.
requirements.txt	Reprodutibilidade. Essencial para Colab e outros ambientes.
.gitignore	Mantém o repositório limpo de arquivos grandes e temporários.
LICENSE	Protege e permite uso aberto. MIT é ideal para projetos educacionais.
