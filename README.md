# 🍽️ AI Food Audit Assistant

> Sistema inteligente para análise automatizada de auditorias de Qualidade e Segurança de Alimentos

## 🎯 Objetivo

Automatizar a análise de auditorias com **Pandas** e **IA local**, gerando relatórios, gráficos e recomendações sem depender de APIs pagas.

Ideal para estudantes, auditores e profissionais de indústria alimentícia.
